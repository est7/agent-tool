# Project Rules Templates (`cfg/templates/rules-templates`)

Project-scoped rule templates intended to be installed into a repository (for example under `.claude/rules/`) via the `agent-tool.sh cfg init rules ...` workflow.

Templates:

- `android.md`: Android project rules.
- `ios.md`: iOS project rules.
- `web.md`: Web project rules.
- `backend.md`: Backend project rules.
- `python-zenml-domain-models.mdc`: domain-specific rules for a Python ZenML models workflow.
